This text file is the instructions of the game.

The runGame.py runs different modes of the game and the other files are just helper files. In runGame.py, please uncomment the line of codes under the name of the mode that you want to play and run it. For instance, if you want to play the 3D Tic Tac Toe 2 player mode, please uncomment the code in line 25, the line under ‘# 2 player mode’ under the section ‘# 3D Tic Tac Toe’. After playing it, please comment back the code.

In the 3D Tic Tac Toe game, the poles that the player puts his token into is according to the mouse position, since this is easier for the player to do so.

In 2D Tic Tac Toe game, to put your token into the box, simply click it.
